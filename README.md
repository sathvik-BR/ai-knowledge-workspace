# AI Knowledge Workspace

An AI-powered application that allows users to **chat with PDF documents**, perform **semantic search**, generate **summaries**, and create **study flashcards** using Retrieval-Augmented Generation (RAG).

---

## Features

* Chat with documents using AI
* Semantic search across PDF content
* Document summarization
* Flashcards generation for studying
* Multi-PDF support
* Conversation history
* Export chat, summaries, and flashcards
* Clean SaaS-style interface

---

## Technologies Used

* Python
* Streamlit
* LangChain
* FAISS Vector Database
* HuggingFace Embeddings
* Groq LLM API

---

## Installation

Clone the repository

```
git clone https://github.com/YOUR_USERNAME/ai-knowledge-workspace.git
```

Install dependencies

```
pip install -r requirements.txt
```

Run the application

```
streamlit run app.py
```

---

## API Key Setup

Replace this line in the code:

```
Groq(api_key="YOUR_GROQ_API_KEY")
```

with your own Groq API key.
